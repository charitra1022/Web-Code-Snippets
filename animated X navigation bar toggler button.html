Animated toggler for navigation bar using bootstrap 4.5

Resize the browser to see the results

Go to the <!-- Animated button code --> section in the code, and notice that in class attrib, there is 'x' class present.

Just add this 'x' class in your button tag, and copy all the <span> codes inside your button tag, and copy all the CSS styles and paste it onto your code's stylesheet.

Make sure that this code uses Bootstrap 4.5 to accomplish the task. If your version is different, then just change the '.navbar-toggler' from '.navbar-toggler.x' in CSS code to some other according to your Bootstrap version.
